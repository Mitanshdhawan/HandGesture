# ✋ Hand Gesture Detection using YOLOv8

## 📌 Project Description
This project implements a real-time hand gesture detection system using the YOLOv8 algorithm. The model is trained to detect hand gestures like fist, one, peace, and palm using a webcam.

---

## 🎯 Features
- Real-time hand gesture detection  
- High accuracy (~95% mAP)  
- Custom-trained YOLOv8 model  
- Multiple gesture recognition  

---

## 🧠 Technologies Used
- Python  
- YOLOv8 (Ultralytics)  
- OpenCV  

---

## 📊 Model Performance
- mAP50: ~95%  
- Precision: ~95%  
- Recall: ~94%  

---

## 🚀 How to Run

Install dependencies:
pip install ultralytics opencv-python

Run project:
python predict.py
